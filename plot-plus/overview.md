# Overview

PlotPlus is a plot management plugin for Hytale servers.

## Features

- Create and manage plot worlds
- Claim and protect plots
- Merge adjacent plots for larger builds
- Trust players or ban them from your plot
- Multi-plot navigation (own multiple plots, teleport with `--number`)
- BuilderTools integration (brush, line tool, paste, scripted brushes)
- Selection tool protection (transform, rotation, bedrock layer)
- Customizable plot templates
- Multi-language support (10 languages, customizable)
- SQLite and MySQL support
- Economy integration
- Developer API with events
